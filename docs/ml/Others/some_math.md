# 一些数学概念

## 差集
$\forall a \in A$ \ $a_1$

代表对于所有的在$A$里面但又不是$a_1$的$a$。

[维基百科:差集](https://zh.wikipedia.org/wiki/%E8%A1%A5%E9%9B%86)

## 范数
范数是具有长度概念的函数。

## 分类与回归
分类问题的输出是离散的，而回归问题的输出是连续的。

::: tip 提示
不要那么轻易就否定自己，知识面前没有主流，没有权威。只有事实。
:::
